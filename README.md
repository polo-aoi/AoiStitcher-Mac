# AoiStitcher for Mac

**从 Python 脚本到 Swift 原生 App 的全面进化。**

AoiStitcher 是一款专为摄影师打造的极简长图拼接工具。在全新的 v2.0.0 版本中，我们舍弃了旧有的 Python 架构，使用 Swift / SwiftUI 进行了底层重构，带来了极致丝滑的交互体验和完全契合 macOS 的原生视觉。

本项目代码与核心逻辑由开发者与 **Gemini** 共同协作完成。

---

## 🌟 核心优势 (Why AoiStitcher?)

* **🪶 极致轻量，仅 3MB 体积** 告别臃肿！得益于纯粹的 Swift 原生开发，软件无需打包庞大的运行环境，体积仅为惊人的 3MB。秒开秒关，不占用宝贵的系统资源。
* **⚡️ 丝滑的原生性能** 直接调用 macOS 底层图形引擎。即使导入数十张高清样片进行长距离拖拽排序，依然能保持满帧的跟手感，彻底告别旧版或网页壳工具的卡顿感。
* **⌨️ “左手键盘，右手鼠标”工作流** 独创的 **QWEAS** 盲操快捷键流。左手掌控全局，右手专注微调。无需在菜单栏来回寻找，出图效率呈倍数提升。
* **🍏 纯正的原生体验** 完美适配 macOS 现代审美。支持原生深色/浅色模式手动切换，提供极致精简的参数面板，让拼接过程回归纯粹。

---

## 🚀 v2.0.0 重大更新

* **底层重构**：基于 SwiftUI 全新开发，响应速度与稳定性质变升级。
* **精准渲染**：重写了导出引擎，解决高分辨率下间距比例失调 Bug，实现 1:1 所见即所得。
* **交互进化**：优化了照片排序算法，引入“视觉快照”逻辑，拖拽更加智能、精准。
* **UI 焕新**：全新的简约扁平化图标，更加融入 macOS 程序坞。

## 🛠 核心功能

* **无极画布缩放**：支持 `Option + 滚轮` 或触控板双指捏合，按住空格键可自由平移画布。
* **独立裁剪工作台**：双击图片进入。支持多种画幅比例预设与旋转纠偏。
* **水印盖印系统**：支持底部全局水印及单张图片的局部水印，可自由调整位置、缩放与角度。

## ⌨️ 快捷键指南 (QWEAS)

专为极致效率设计，左手无需离开键盘核心区：

* `[A]`：添加照片 (Add)
* `[Q]`：全局预览自适应大小 (Quick View)
* `[S]`：选择全局水印 (Select Watermark)
* `[W]`：清空当前画布 (Wipe)
* `[E]`：导出最终长图 (Export)

## 📦 安装与使用

1.  前往 [Releases](../../releases) 页面下载最新的 `AoiStitcher.zip`。
2.  将解压后的 `AoiStitcher.app` 移动至 **应用程序 (Applications)** 文件夹。
3.  **首次运行**：因本地打包未配置开发者证书，请**按住 Control 键并右键点击**图标，选择“打开”，在弹出的对话框中再次点击“确认打开”即可。
