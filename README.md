# AoiStitcher for Mac

**从 Python 脚本到 Swift 原生 App 的全面进化。**

AoiStitcher 是一款专为摄影师打造的极简长图拼接工具。在全新的 v2.0.0 版本中，我们舍弃了原有的 Python 架构，采用 Swift / SwiftUI 进行了底层重构，带来了极致丝滑的交互体验和完全契合 macOS 的原生视觉。

本项目代码与核心逻辑由开发者与 **Gemini** 共同协作完成。

---

## ✨ v2.0.0 重大更新

* **原生重构，性能飞跃**：告别旧版长距离拖拽时的卡顿，海量高清样片实时排序，零延迟响应。
* **极客级快捷键工作流**：专为“左手键盘，右手鼠标”设计的 **QWEAS** 盲操快捷键流，极大提升出图效率。
* **全新 UI 与深色模式**：重绘了更符合现代审美的扁平化 App 图标，完美适配 macOS 原生深色/浅色主题。
* **精准渲染引擎**：修复了旧版高分辨率导出时的间距比例失调问题，实现预览与导出的 1:1 所见即所得。

## 🚀 核心功能

* **无极画布缩放**：支持 `Option + 滚轮` 或触控板双指捏合缩放，按住空格键可自由平移画布。
* **丝滑拖拽排序**：直接拖动照片即可重新排列顺序，底层算法自动计算绝对居中并吸附对齐。
* **独立裁剪工作台**：双击单张图片进入裁剪模式，支持多种常见画幅比例预设与无极旋转纠偏。
* **全局与局部水印**：支持添加底部全局水印，或在单张图片编辑模式内添加局部水印并自由调整缩放与角度。

## ⌨️ 快捷键指南 (QWEAS)

为极致效率设计，左手无需离开键盘核心区：

* `[A]`：添加照片 (Add)
* `[Q]`：全局预览自适应大小 (Quick View)
* `[S]`：选择 / 更换全局水印 (Select Watermark)
* `[W]`：清空当前画布全部照片 (Wipe)
* `[E]`：导出最终长图 (Export)

## 📦 安装与使用

1. 前往本仓库的 [Releases](../../releases) 页面下载最新的 `AoiStitcher.zip`。
2. 解压缩后，将 `AoiStitcher.app` 拖入你的 **应用程序 (Applications)** 文件夹中。
3. **首次运行注意事项**：因本地打包未配置苹果开发者证书，直接双击可能会被系统拦截。请**按住键盘上的 `Control` 键，同时用鼠标右键点击**软件图标，在菜单中选择**“打开”**，然后在弹出的警告框中再次点击“确认打开”即可。以后便可正常双击运行。
